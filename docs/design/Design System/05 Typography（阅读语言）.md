# BioassayCraft Design System

# 第五章｜Typography（阅读语言）

> Typography 不是选择字体。
>
> Typography 是组织阅读。

---

# 为什么 Typography 如此重要

BioassayCraft 的核心不是动画。

不是 D3。

不是交互。

而是：

阅读。

用户的大部分时间，

都会用于：

阅读问题。

阅读推理。

阅读批注。

阅读观察。

因此，

Typography 决定了：

整个产品的学习体验。

---

# Typography 的目标

字体存在，

不是为了品牌。

而是为了：

减少阅读阻力。

帮助用户思考。

优秀 Typography，

应该让用户忘记：

字体。

而只关注：

科学内容。

---

# 阅读优先于设计

任何视觉决策，

都不能降低阅读体验。

因此：

漂亮，

永远排在：

可读性

之后。

---

# 建立固定层级

整个网站，

永远保持固定层级。

推荐：

Level 1

Scientific Question

页面真正的问题。

例如：

How do we know this assay is suitable?

整个 Scene，

只能存在一个。

---

Level 2

Core Statement

一句最重要的话。

例如：

ATP defines the intended decision.

不是标题。

而是：

这一 Scene 最重要的思想。

---

Level 3

Explanation

正文。

解释：

为什么。

正文应保持：

简短。

连续。

避免大段文字。

---

Level 4

Margin Note

Notebook 批注。

长度：

通常不超过：

一到两行。

例如：

"Evidence first."

"Think again."

"Model before statistics."

Margin Note

永远短于正文。

---

Level 5

Reference

法规。

公式。

说明。

属于背景信息。

不应与正文竞争。

---

# 一屏只有一个标题

不要：

Hero

↓

Journey

↓

Tool

↓

Notebook

四个标题。

任何 Scene，

只有一个真正标题。

其余内容，

都属于 Supporting Text。

---

# 控制每行长度

正文不要过宽。

推荐：

60~80 个字符。

太宽：

阅读疲劳。

太窄：

阅读破碎。

BioassayCraft 更接近：

书籍。

而不是：

新闻网站。

---

# 控制段落长度

正文，

尽量保持：

3~5 行。

不要：

十几行连续文字。

长内容，

应拆分：

Observation

↓

Reasoning

↓

Conclusion

---

# 每一段只表达一个思想

不要：

一段同时解释：

Bias。

Precision。

Accuracy。

Sample Size。

每段：

一个思想。

这是：

Scientific Narrative

最重要的原则。

---

# Margin Note 比正文更重要

很多网页，

正文最大。

BioassayCraft 不一样。

真正帮助用户思考的，

往往是：

Margin Note。

例如：

正文：

解释：

为什么 ATP。

Margin：

"Purpose before method."

用户最后记住的，

应该是：

Margin Note。

---

# 不要强调结果

不要：

Pass

Fail

Correct

Wrong

大面积加粗。

真正应该强调的是：

Question。

Reasoning。

Evidence。

结果，

通常应保持：

克制。

---

# Typography 应表达节奏

阅读，

应存在：

停顿。

例如：

Question

↓

留白

↓

Core Statement

↓

留白

↓

Explanation

↓

Margin Note

不要：

文字连续堆积。

---

# 引用应该安静

法规：

USP

ICH

EP

不应成为：

视觉中心。

Reference

始终属于：

背景层。

真正重要的是：

Scientific Meaning。

不是：

法规编号。

---

# 字体数量

整个网站，

建议：

最多三种字体。

Logo

正文

数字

除此之外，

不要增加字体。

一致性，

远比丰富重要。

---

# 数字必须等宽

所有：

统计量。

P 值。

CI。

Residual。

Dose。

Concentration。

建议：

使用等宽字体。

方便：

比较。

观察。

推理。

---

# Typography Checklist

任何页面，

发布之前，

回答：

□ 是否只有一个 Scientific Question？

□ 是否建立固定层级？

□ 是否控制正文长度？

□ 是否控制每行宽度？

□ Margin Note 是否短于正文？

□ 是否避免长段文字？

□ 是否保持大量留白？

□ 是否突出 Question 而不是 Result？

□ 是否减少字体种类？

□ 是否帮助用户连续阅读？

如果不能全部满足，

Typography

继续修改。

---

# 本章总结

Typography，

不是页面装饰。

它是一种阅读设计。

真正优秀的 Typography，

不会让用户记住：

字体。

字号。

加粗。

而会让用户：

一直保持思考。

这就是 BioassayCraft Typography 的目标。
