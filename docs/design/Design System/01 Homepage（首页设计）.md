# BioassayCraft Design System

# 第一章｜Homepage（首页设计）

> 首页不是产品介绍。
>
> 首页是一段科学旅程的开始。

---

# 首页的职责

BioassayCraft 首页只有三个目标。

第一：

建立品牌。

第二：

建立产品哲学。

第三：

邀请用户开始思考。

除此之外，

首页不承担任何职责。

它不是：

- 导航页
- 产品介绍页
- 功能列表
- 更新日志
- Tool Catalog

首页不是为了告诉用户：

这里有什么。

而是为了告诉用户：

这里将如何思考。

---

# Hero 只完成一件事情

Hero 永远只能回答一个问题：

BioassayCraft 是什么？

因此，

Hero 应始终保持极简。

推荐结构：

BIOASSAY REASONING NOTEBOOK

↓

BioassayCraft

↓

From theory to practice.

↓

一句简短说明

↓

Scroll to begin

Hero 中禁止出现：

Journey

USP

Tool

Feature

Calculator

Simulator

Validation

Method Development

首页第一屏，

永远不要开始教学。

第一屏，

只建立品牌。

---

# 不要出现 CTA

BioassayCraft 不需要：

Start Now

Explore

Get Started

Try It

Book Demo

Learn More

这些按钮属于 SaaS。

BioassayCraft 希望用户：

自然开始阅读。

因此：

Scroll

就是最好的 CTA。

---

# 第一屏不要出现产品结构

Hero 中禁止出现：

Journey

Learn

Tools

三个栏目。

用户第一次进入网站，

不需要知道网站结构。

用户首先需要知道：

为什么值得继续向下阅读。

结构，

应该逐渐形成。

而不是一次展示。

---

# Hero 不应该解释 Bioassay

Hero 不负责：

介绍 Bioassay。

也不负责：

介绍 USP。

更不负责：

介绍 Validation。

Hero 只负责：

建立一种安静、

可信、

克制、

值得继续阅读的第一印象。

---

# Hero 背景

Hero 背景，

应接近空白。

允许：

极浅 Notebook Grid

极浅 Coordinate Paper

极弱呼吸动画

极弱噪声纹理

禁止：

粒子背景

知识网络

漂浮文字

Random Node

发光线条

大型 D3 Graph

Hero 的背景，

永远不能比标题更吸引注意。

---

# Hero 的留白

Hero 最大的元素，

不是 Logo。

而是：

留白。

页面四周，

都应保持充足呼吸空间。

不要为了填满屏幕，

增加元素。

宁可空。

不要满。

---

# Hero 字体

Hero 只有四级层级。

Level 1

Brand

BioassayCraft

Level 2

Slogan

From theory to practice.

Level 3

Description

一句完整说明。

Level 4

Scroll Hint

不要增加第五级。

不要增加副标题。

不要增加标签。

---

# Hero 的动画

Hero 动画，

必须几乎感觉不到。

允许：

Fade

Opacity

Very Slow Translation

禁止：

Scale

Rotation

Bounce

Parallax

Hero 的动画，

应该让用户感觉：

页面自己醒来了。

而不是：

开始表演。

---

# Hero 结束方式

Hero 不应该：

突然切换。

Hero 应：

自然淡出。

随后：

Question

自然进入。

Brand

慢慢退到背景。

Journey

开始形成。

Hero 应：

成为 Journey 的起点。

而不是：

结束。

---

# Homepage 是 Narrative

首页，

不是多个 Section。

首页，

是一段 Narrative。

因此：

Screen1

↓

Screen2

↓

Screen3

↓

Screen4

必须保持：

连续。

任何两个 Screen，

都必须回答：

为什么进入下一屏？

如果回答不了，

说明 Narrative 断裂。

---

# Homepage 永远只有一个主角

首页任何时刻，

只能存在一个主角。

例如：

第一页：

Brand。

第二页：

Question。

第三页：

ATP。

第四页：

Journey。

第五页：

Evidence。

第六页：

Tool。

不要：

Brand

Journey

Tool

Notebook

同时出现。

---

# Homepage Checklist

任何 Homepage 修改，

发布之前，

必须回答：

□ Hero 是否只完成一件事情？

□ 是否没有 CTA Button？

□ 是否没有 Tool Grid？

□ 是否没有 USP 标签？

□ 是否保持大量留白？

□ 是否只有一个视觉中心？

□ 是否遵循 Scientific Narrative？

□ 是否自然进入 Journey？

□ 是否体现 Notebook？

□ 是否符合 From theory to practice？

如果任何一项答案是否定，

首页不得发布。

---

# 本章总结

优秀的首页，

不会告诉用户：

产品有什么。

而会让用户产生：

继续思考的欲望。

BioassayCraft 首页，

不是 Landing Page。

它是一本科学笔记的第一页。

真正重要的，

不是第一页讲了多少内容。

而是用户是否愿意继续翻开第二页。
