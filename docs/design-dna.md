# BioassayCraft Design DNA

We reduce before we decorate.

We explain before we calculate.

We reveal before we automate.

We guide before we judge.

We prefer understanding over feature count.

We value reasoning over memorization.

We build journeys instead of isolated tools.

We earn trust through clarity.

We respect scientific uncertainty.

We craft slowly.

Every BioassayCraft artifact should help users think more clearly than before.
