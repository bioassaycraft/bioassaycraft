Interpretation

Formula details

USP <1033>

Prob(OOS) = Phi(ZL) + 1 - Phi(ZU)

ZL = [ln(LSL) - RBlog] / sigma

ZU = [ln(USL) - RBlog] / sigma

ChP 9401

P(OOS) is evaluated from ACRB and ACRVV on log scale.

ACRBlog = ln(1 + ACRB / 100)

sigma = ln(1 + ACRVV / 100)
