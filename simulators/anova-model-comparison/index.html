ANOVA Explorer

This page helps you understand how model assumptions change residuals, variance decomposition, and ANOVA decisions in bioassays.

Scientific question

When a model becomes more flexible, which part of the residual variation disappears, and what does that change mean for similarity, linearity, and model fit?

Scientific idea

ANOVA is not just a table of p-values. It is a way to ask which model assumption explains variation, and which unexplained variation should still make us cautious.

Mentor note

Watch the residuals before reading the table. The table is easier to interpret after you can see what each model assumption removes from the data.

Scope of use

This explorer teaches model-comparison reasoning through linear, SRA, PLA, and 4PL examples. It is intended to make residuals, sum-of-squares decomposition, and ANOVA decisions easier to reason about, not to replace formal assay-specific statistical review.
